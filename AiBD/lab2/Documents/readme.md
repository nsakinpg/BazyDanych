# Data
Here are described all files from AnalysisData folder

## drinks.csv
Original file:
* country - name of the country
* beer_servings - cans of beer drunk per person in country
* spirit_servings - shots of spirits drunk per person in country
* wine_servings - glasses of wine drunk per person in country
* total_litres_of_pure_alcohol - number of total litres of pure alcohol consumed by person in country

## drinks_p.csv
To this file was added column total_servings which contains all beer, wine and spirits serving in country.
